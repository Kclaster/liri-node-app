# liri-node-app

#Liri Nod App

Liri is an app designed to retrieve data from various api's given a command. 
The commands are as follows:
*concert-this
*spotify-this-song
*movie-this
*do-what-it-says

## concert-this
The concert-this command retrieves access to a venue, location, and date of an event.
